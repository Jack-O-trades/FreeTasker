# 🚀 FreeTasker

FreeTasker is an AI-powered freelancing platform designed to create a fair, transparent, and secure environment for both freelancers and clients.

## ✨ Features

* 🔐 Role-based authentication (Client, Freelancer, Admin)
* 💼 Project posting and management
* 💰 Smart tier-based bidding system (Basic, Standard, Premium)
* 🤖 AI-generated proposals and project descriptions
* 📊 Bid transparency (Viewed, Shortlisted, Rejected, Ignored)
* 💬 Real-time chat using WebSockets
* ⭐ Rating and reputation system
* 🏆 Achievement badges and priority rankings
* 🛡️ Anti-spam, anti-scam, and anti-ghosting mechanisms
* 💳 Secure payment and escrow workflow
* 🎯 Skill-based project matchmaking
* 👨‍⚖️ Admin dashboard for dispute resolution and moderation

## 🛠️ Tech Stack

**Frontend:** React.js, Axios
**Backend:** Django, Django REST Framework
**Database:** PostgreSQL
**Real-Time Communication:** Django Channels, Redis, WebSockets
**AI Integration:** OpenAI API

## 🎯 Vision

To build a transparent, trustworthy, and AI-assisted freelancing ecosystem where clients find the right talent and freelancers get fair opportunities.
